1) To create an executable for OpenMP use command 
   make openmp
2) To create an executable for PThread use command 
   make pthread
3) Format for execution
   ./openmp 2 ~/Downloads ~/Downloads
   ./pthread 2 ~/Downloads ~/Downloads
   File_to_be_executed number_of_filesystems absolute_path_of_filesystem_1 ... absolute_path_of_filesystem_n
4) Output gives the list of Common Files and Folders as well as tells whether filesystems are divergent or exactly same.
